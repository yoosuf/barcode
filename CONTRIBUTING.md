# Contributing

## Development Setup

1. Clone repository.
2. Install dependencies with Composer.
3. Use path repository integration in host Laravel app for local testing.

## Coding Guidelines

- Keep public APIs backward compatible in minor releases.
- Follow PSR-12 style and package namespace conventions.
- Keep drivers focused and stateless.

## Pull Request Checklist

- Updated docs in README and docs folder.
- Added/updated tests for behavior changes.
- No breaking API changes without major version bump.
- Updated changelog for user-facing changes.

## Release Notes

Document migration and compatibility notes for any change that could impact integrators.
